보안 솔루션

엔드 디바이스부터 클라우드까지, 모든 계층을 보호합니다

TSA

Trusted Safety for Multiple AI Agents

다수의 AI 에이전트의 상호작용에서 발생하는 예상치 못한 위험을 조절하고 제어하는 통합형 프레임워크입니다.

  • 통합형 멀티 에이전트 프레임워크 구축
  • 에이전트 입출력 보호 - 민감정보, 윤리적 기준 위반 방지
  • 에이전트 모델 보호 및 DRM 라이선스 관리
  • 모델 탈취 방지 및 외부 노출 차단
AI Agent Safety Multi-Agent System

TPA

Trusted Privacy Filter Agent

AI 서비스 이용 시 사용자의 개인정보와 기밀정보를 가명화하여 안전하게 보호하는 프라이버시 필터 에이전트입니다.

  • 개인 식별 정보(PII) 및 영업비밀 자동 탐지 및 가명화 처리
  • AI 기반 맥락 분석으로 정보 의미를 유지한 채 가명화 수행
  • 웹 브라우저 확장 프로그램으로 다양한 AI 서비스에 대응
  • Membership inference, model inversion 등 다양한 공격 방어
Privacy Protection PII Filtering

DPF

Dialect Perimeter Filter

고유한 Dialect 기반 통신 체계를 활용하여 네트워크 경계를 보호하는 차세대 보안 시스템입니다.

  • Dialect 기반 디바이스 인증 및 제로 트러스트 구현
  • Dialect of Things - IoT 환경 네트워크 보안
  • 운영체제/커널 레벨 네트워킹 보안
  • 원격 메모리 시스템 보안 연구 기반
Network Security Zero Trust IoT Security

CC-BOX

Confidential Computing Platform

데이터 프라이버시 및 보안 규제 준수를 위한 기밀성 보장 보안 플랫폼입니다. 저장 데이터뿐만 아니라 연산 중인 데이터까지 안전하게 보호합니다.

  • 연산 중 데이터 보호 - 실행 시에도 기밀성 보장
  • 금융/의료/기업 비밀 데이터의 규제 준수 지원
  • 지적 재산(IP)을 서비스 플랫폼 제공자로부터 보호
  • 간편한 적용으로 넓은 시스템 범위 지원
Confidential Computing Data Privacy Compliance

CC-SBOM

Confidential Computing SBOM

CC-BOX 안전 실행 환경과 연계하여 소프트웨어 구성 명세서(SBOM) 생성을 지원하는 보안 솔루션입니다.

  • 사용자 기밀정보(소스코드 등) 유출 방지
  • 실시간 취약점 관리 서비스 제공
  • 클라우드 및 서버 소유자로부터 소프트웨어 자산과 IP 보호
  • 의료, 금융, OEM, 산업 제어 시스템 등 규제 분야에 특화
SBOM Supply Chain Security Vulnerability Management

Outerstellar

Hardware-based Instruction-flow Safety Monitor

하드웨어 기반으로 시스템의 명령어 흐름을 모니터링하여 무결성을 보장하는 보안 솔루션입니다. ACM CCS에 게재된 Interstellar 연구를 기반으로 합니다.

  • 하드웨어 수준의 명령어 흐름 모니터링
  • OS 커널 무결성 실시간 검증
  • 소프트웨어 조작이 불가능한 하드웨어 기반 보안
  • Top-Security Conference(ACM CCS) 검증 기술
